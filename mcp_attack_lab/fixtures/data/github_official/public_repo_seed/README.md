# gh-toxic-flow-public-lab

This repository is a controlled public testbed for MCP toxic-flow experiments.

It contains only harmless toy code and experiment scaffolding.

Purpose:

- host a public issue that contains indirect prompt injection content
- act as the public sink for issue comments or draft pull requests
- provide a realistic but low-risk repository for official GitHub MCP testing

This repository should never contain:

- real secrets
- proprietary code
- personal data
- third-party confidential material
